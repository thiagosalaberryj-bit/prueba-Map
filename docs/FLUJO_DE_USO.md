# Cómo usar la app paso a paso

## Modo Marcador

```
1. Abrí la app → el mapa se centra en tu ubicación GPS
2. Tocá el botón "📍 Marcador" (abajo, a la izquierda)
3. Opción A: Escribí una dirección en el buscador → tocá el resultado
   Opción B: Mantené presionado cualquier lugar del mapa
4. Aparece un pin azul en esa ubicación ✅
```

**SafeBike**: Marcá dónde dejaste tu bici, o puntos de interés.

---

## Modo Ruta (por calles)

```
1. Tocá "🛤️ Ruta" abajo
2. Agregá puntos:
   - Buscá una dirección → tocá el resultado
   - O tocá "🎯 Agregar punto actual" para usar tu GPS
3. Repetí para agregar el destino (y paradas si querés)
4. Cuando tengas 2+ puntos, tocá "🚗 Obtener ruta"
5. ¡La ruta se dibuja SIGUIENDO LAS CALLES! ✅
```

**SafeBike**: Planeá tu recorrido en bici y conocé la distancia exacta.

---

## Modo Zona de riesgo

```
1. Tocá "⚠️ Zona" abajo
2. Elegí el radio (100, 200, 500, 1000, 2000 metros)
3. Buscá una dirección o mantené presionado el mapa
4. Aparece un círculo rojo con ese radio
```

**SafeBike**: Marcá zonas peligrosas o robos frecuentes para tener precaución.

---

## ⏺ Grabar recorrido (GPS)

```
1. En cualquier modo, tocá "⏺ Grabar ruta"
2. Empezá a caminar o andar en bici
3. Se dibuja una línea naranja en tiempo real 
4. Cuando termines, tocá "⏹ Detener grabación"
5. La ruta se guarda automáticamente 
```

**SafeBike**: Registrá tu viaje completo para después analizarlo.

---

## 📋 Lista de elementos

```
1. Tocá "📋 N" (abajo a la derecha) para ver todos los elementos
2. Cada elemento muestra su tipo (📍 🛤️ ⚠️) y nombre/dirección
3. Tocá 🗑 al lado de un elemento para eliminarlo
4. Tocá ✕ arriba para cerrar la lista
```

---

## 📍 Botón de ubicación

```
El botón "📍" (esquina derecha del mapa) centra el mapa en tu posición actual.
```

## 🗑 Limpiar todo

```
El botón "🗑 Limpiar" borra TODOS los elementos. Te pide confirmación antes.
```
